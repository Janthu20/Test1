Machine Design PLC Programming ESP32 IoT Automation

Overview

During my internship at LTL Transformers (Pvt) Ltd, I was immersed in the real-world engineering challenges of a high-capacity manufacturing facility. My role focused on identifying inefficiencies in the production line and designing automated solutions to address them.

Key Responsibilities & Contributions

  • Machine Design: Designed and modified machinery components using SolidWorks to improve production workflow. This included the design of a specialized wire tensioning arm for transformer winding.
  • Automation Systems: Developed semi-automatic solutions for manual tasks, such as the Yoghurt Filling Machine project, which utilized pneumatic logic and Xinje PLC control.
  • PLC Programming: Gained extensive hands-on experience writing Ladder Logic for Xinje and Siemens PLCs, integrating sensors, actuators, and HMI panels.
  • IoT Development: Designed an ESP32-based controller for monitoring oil tank levels. This system used ultrasonic sensors and web-based monitoring to automate valve operations.

Skills Acquired

This training bridged the gap between theoretical mechatronics and practical application. I learned to troubleshoot industrial electrical panels, select appropriate pneumatic components (cylinders, valves), and manage project timelines within a production environment.
